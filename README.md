# Jimotmee..
너의 추억.. 지켜주지 못해서 미안해..

앞으로는 지못미가 지켜줄게!!

지못미는 예전의 추억을 소환하여 기록하고 공유할 수 있는 웹페이지 입니다

<br>

## 아이디어 빌딩
>"기록해두지 못한 어렸을 때의 추억을 불러올 순 없을까?"
>
>"인터넷이 지금처럼 흔하게 쓰이지 않았을 시절을 지금의 인터넷으로 구현할 순 없을까?"
>
>하는 생각에서 시작했습니다
